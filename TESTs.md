### ðŸ”¸ 1. **Enroll a Student**

**Endpoint:** `POST /api/enrollments`

```bash
curl -X POST http://localhost:8080/api/enrollments \
  -H "Content-Type: application/json" \
  -d '{
    "studentId": 1,
    "courseId": 1,
    "paymentStatus": "PENDING",
    "paymentMethod": "PAYPAL"
}'
```
